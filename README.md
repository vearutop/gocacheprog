A work in progress to implement GOCACHEPROG tools for large codebases.

## Install

```
go install github.com/vearutop/gocacheprogd@latest
$(go env GOPATH)/bin/gocacheprogd --help
```

Or download binary from [releases](https://github.com/vearutop/gocacheprogd/releases).

### Linux AMD64

```
wget https://github.com/vearutop/gocacheprogd/releases/latest/download/linux_amd64.tar.gz && tar xf linux_amd64.tar.gz && rm linux_amd64.tar.gz
./gocacheprogd -version
```


## Usage

Create a new repository from this template, check out it and run `./run_me.sh` to replace template name with name of
your repository.
